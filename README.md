# React App with Vite

This is a React application built using Vite.

## Prerequisites

- Node.js (version 14 or above recommended)
- npm or yarn package manager

## Getting Started

1. **Clone the repository**

```bash
git clone <repo-url>
cd <repo-folder>
```

2. **Install dependencies**

```bash
npm install
```

3. **Start the development server**

```bash
npm run dev
```
